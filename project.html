五山校区周边餐饮分析

华南理工大学五山校区 · 2000m 半径 · 全品类餐饮 POI
高德地图 POI 采集 + DeepSeek 智能分类 + Folium 交互可视化
GitHub 仓库 全屏地图
项目流程
1

网格化采集

5×5 网格（25 个子区），覆盖华南理工大学五山校区周边 2000m 范围。高德 API 分页获取，遇限流自动重试（指数退避）。按店名+地址去重。

2

智能分类

DeepSeek 大语言模型根据店名和地址，自动归类到 10 个餐饮类别（火锅、茶饮咖啡、粉面小吃、中餐正餐、异国料理等），含边界案例 few-shot 示例。批量调用每 10 条合并一次请求，结果缓存 JSON，重复运行零费用。

3

交互式地图

Folium 生成 Leaflet 地图，支持图层控制（10 个类别 + 热力图自由开关）、MarkerCluster 聚合标记（显示数量，点击展开）、彩色圆点标记、悬停显示店名、点击弹窗含高德导航链接。

4

统计图表

Matplotlib 绘制类别数量柱状图 + 占比饼图，自动适配中文字体（SimHei）。控制台输出分类摘要：总数、Top 类别、茶饮/火锅数量等。
