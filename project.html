GitHub Repo Full Map
Pipeline
1

POI Collection

5x5 grid (25 cells) via Amap API covering 2000m radius around SCUT Wushan campus. Auto-retry on rate-limit with exponential backoff.

2

Smart Classification

DeepSeek LLM classifies shops into 8 categories (hotpot, tea, noodles, etc.) with few-shot prompting. Results cached to JSON for zero-cost re-runs.

3

Interactive Map

Folium + Leaflet with LayerControl, MarkerCluster, HeatMap. Color-coded per category. Popup with Amap navigation link. Legend panel with collection date.

4

Charts & Report

Matplotlib bar + pie charts. Console summary with top categories and distribution insights.
